# Computer Vision Bootcamp 🚀

This repository contains multiple **real-time Computer Vision experiments** built using **Python, OpenCV, and MediaPipe**.
The project demonstrates how to use a webcam to apply filters, detect faces, and track body movements.

---

## 📌 Features

*  Body Pose Tracking
*  Face Mesh Detection (468 face landmarks)
*  Live Webcam Filters
*  Blur Effect
*  Edge Detection
*  Thermal Camera Simulation
*  Heatmap Visualization
*  Zoom Camera Effect
*  Live Text Overlay

All programs run **in real-time using your webcam**.

---

## 🛠 Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy

---

## 📂 Project Structure

cv-bootcamp/

body_tracking.py # Real-time body pose detection
face_mesh.py # Face landmark detection

live_MNV.py # Motion noise visualization
live_text.py # Text overlay

live_zoom.py # Zoom effect

## ⚙️ Installation

### 1️⃣ Clone the repository

git clone https://github.com/devashreep22/cv-bootcamp.git
cd cv-bootcamp

### 2️⃣ Create virtual environment (optional)

python -m venv cv_bootcamp_env

### 3️⃣ Activate environment

Windows:

cv_bootcamp_env\Scripts\activate

### 4️⃣ Install dependencies

pip install opencv-python mediapipe numpy

---

## ▶️ Run the Project

Example:

python face_mesh.py

or

python body_tracking.py

Press **ESC** to exit the webcam window.
